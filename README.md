## Description

This is a Todo App made using NestJS starter boilerplate. The PostgreSQL database is running on a docker container. A sample env variable file has been provided in the repository by the name of `env.sample`. Before running the app, either rename the env file to `.env` or make a new `.env` file with your credentials.

However please note, you would need to adjust your Postgres credentials in the `docker-compose.yml` file as well should you chose to do that.

## Running the app

1. Install the packages
2. Start the docker container
3. Run the app

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# docker
$ docker compose up

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

# prisma docs
$ yarn prisma-docs-generator serve
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
